# 🚀 AttentionX — AI Content Repurposing Engine

## 📌 Problem
Long-form content (lectures, podcasts, workshops) contains valuable insights, but modern audiences prefer short-form videos (Reels, Shorts).

Manually finding "viral moments", cropping, and adding captions is time-consuming.

---

## 💡 Solution
AttentionX is an AI-powered platform that converts long-form videos into short, engaging clips automatically.

---

## ⚙️ Features
- 🎯 Emotional Peak Detection (Audio + Sentiment simulation)
- 🎬 Smart Vertical Cropping (9:16)
- 🧠 AI Clip Selection (Top viral moments)
- 📝 Auto Captions + Hook Headlines
- 📊 Processing Pipeline Visualization
- 📥 Downloadable clips

---

## 🧪 Current Implementation (MVP Prototype)
This project is a **frontend MVP prototype** simulating the full AI pipeline:

- Upload video + preview
- Configure AI settings
- Simulated processing steps
- Generated clips with scores & hooks

---

## 🔮 Future Scope (Backend Integration)
- 🎙️ Whisper → Speech-to-text transcription

- 🎥 FFmpeg → Clip generation
